# Reusable functions (optional)

Place shared Python modules here and install as a wheel or add to cluster path so notebooks can import, e.g.:

- `ingestion_utils.py`: ingestion_date_utc(), batch_id(), raw_path()
- `quality_checks.py`: run_expectations()
- `logging_utils.py`: log_to_azure_monitor()

Use Repos or cluster init script to attach this folder to the Python path.
